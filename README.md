# deep-learning-playground

一个面向学习和实践的深度学习示例仓库，包含从零实现、实验笔记、Jupyter notebook 示例和小型项目。

**目标**

- 通过动手实现核心算法（从线性回归到神经网络），加深对深度学习原理的理解。
- 提供清晰、可运行的示例和笔记，帮助初学者快速上手。
- 建立可复现的小实验环境，便于对比激活函数、优化器等设计选择。


**快速开始**

1. 克隆仓库（本地已在工作区，则跳过）

```bash
git clone https://github.com/<your-username>/deep-learning-playground.git
cd deep-learning-playground
```

2. 建议使用虚拟环境并安装依赖：

```bash
python -m venv .venv
# Windows
.venv\Scripts\activate
# macOS / Linux
# source .venv/bin/activate

pip install -r requirements.txt
```

3. 运行纯 Python 示例：

```bash
python examples/linear_regression.py
```
